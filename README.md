# フォント比較アプリ MVP

## アプリ概要

複数のフォントを、同じ文章・同じ表示条件で並べて比較できるWebアプリです。見た目、文字幅、字形のバランス、用途や属性をカード形式で確認できます。

## 主な機能

- 左側のチェックボックスによるフォント選択
- フォントカードの一覧表示
- 文字サイズの変更
- 太さの変更
- 行間の変更
- 字間の変更
- 通常表示
- 固定マス表示
- 実幅枠表示
- 詳細表示
- レスポンシブ表示

## 使用方法

フレームワークやビルド作業は不要です。Windows環境のVS Codeやエクスプローラーから、index.html をブラウザで開いて確認してください。

ローカルファイルとして開いた場合、JavaScriptの実行自体は可能ですが、将来の機能追加時にローカル実行制限が生じる可能性があります。

## フォント表示に関する注意

- 閲覧環境に対象フォントがインストールされていない場合、別フォントへフォールバックする可能性があります。
- フォントファイル自体はこのリポジトリに含めていません。
- カードに表示する対応言語や属性の一部は、MVP段階では手動定義です。
- 現段階ではPythonやcmapによる自動解析結果を画面へ連携していません。
- Web利用可否やライセンスは未確認項目を含みます。

## 確認方法

1. index.html をブラウザで開く
2. 初期状態で3種類のカードが表示されることを確認する
3. サイドバーのチェックを切り替え、カードの表示が変わることを確認する
4. 文字サイズ・太さ・行間・字間・表示モードを切り替えて確認する
5. 画面幅を狭くして、2列または1列へ変化することを確認する
